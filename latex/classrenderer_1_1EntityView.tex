\doxysubsection{Detailed Description}
View class representing the visual aspect of an entity in the game. Inherits from \doxylink{classrenderer_1_1DrawObject}{Draw\+Object} to provide drawing capabilities. When inheriting create corresponding Entity\+Model class to handle logic side.

Has a size property to define the dimensions of the entity in the game world.
