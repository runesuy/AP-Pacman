Detailed Description

Factory interface for creating entity models. Provides methods to create different types of entities.

Note
This is an interface and should be implemented by concrete factory classes.
This factory must be passed as logic::Config
On creation of an entity, it should be fully initialized except for position and size.
Each method should also initialize any necessary components for the entity on the representation side.
